# Multi-Page Dynamic Site - Task

This project demonstrates:

- Dynamic content loading from `data/posts.json`
- External API integration (quotes)
- Sticky navbar with scrollspy and smooth-scrolling
- Real-time contact form validation and simulated submit
- Scroll reveal animations and responsive design

## Run locally
1. Open the folder in VS Code
2. Install Live Server extension
3. Right click `index.html` -> Open with Live Server

## Files
- index.html, about.html, services.html, contact.html
- css/styles.css
- js/utils.js, js/main.js
- data/posts.json
